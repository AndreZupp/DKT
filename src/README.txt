This folder contains the essential files to run the DKT architecture.

Each experiment can be run using the respective script:

Experiment1 -----> cifar100_training.py
Experiment2 -----> splitcifar100_pretrained.py
Experiment3 -----> step_training.py 

The source files of the experiments are in the "src" folder.
The folder "runs" provides the essential pre-trained models to run the various experiments.
